## Key Features in Action

### 1. Visual Indicators
- **[âœ“]** = Enabled option
- **[ ]** = Disabled option
- **[M]** = Module (tristate)
- **âš™ï¸** = Configuration option
- **ğŸ“** = Menu with subitems
- **Cyan highlight** = Selected item
- **Gray text** = Disabled item

### 2. Navigation Flow
1. Start at main menu
2. Use â†‘/â†“ to select an item
3. Press Enter to enter a submenu
4. Press ESC to go back
5. Press Space to toggle options

### 3. Search Workflow
1. Press `/` to activate search
2. Type search query (e.g., "debug")
3. Results filtered in real-time
4. Navigate filtered results
5. Press Enter or ESC to exit search

### 4. Modification Tracking
- Header shows "Changed: N" counter
- Modified options tracked
- On quit, prompts to save if changed
- Press 's' to save anytime

### 5. Context-Aware Help
- Right panel updates as you navigate
- Shows symbol type, ID, status
- Displays help text from Kconfig
- Shows dependencies when applicable

## Advantages Over Traditional Menuconfig

1. **Modern UX**: Clean, emoji-enhanced interface
2. **Live Search**: Fuzzy search as you type
3. **Better Navigation**: Vim keys + arrow keys
4. **Rich Context**: Detailed help panel always visible
5. **Visual Clarity**: Icons and colors improve scanning
6. **Fast Performance**: Instant response times
7. **Intuitive**: Easy to learn, efficient to use

## Example Session

```bash
$ cd examples/sample_project
$ rkconf menuconfig

# TUI launches showing configuration options
# Navigate with arrow keys
# Press Space to toggle DEBUG option
# Press 's' to save
# Press 'q' to quit

$ cat .config
# Automatically generated file; DO NOT EDIT.
HAVE_ARCH=y
DEBUG=y
# VERBOSE is not set
```

## Technical Details

- Built with **ratatui** (modern Rust TUI framework)
- Uses **crossterm** for terminal manipulation
- Fuzzy search with custom scoring algorithm
- Efficient rendering with minimal redraws
- Full integration with existing Kconfig parser
